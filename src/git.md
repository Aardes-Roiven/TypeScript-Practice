# Git management

### git add
* `git add .` - добавить все в индекс
* `git add file1.txt` - добавить конкретный файл в индекс
* `git add *.txt` - добавить файлы с конкретным расширением в индекс

### git rm
* `git rm -r .` - удалить 
* `git add file1.txt` - добавить конкретный файл в индекс
* `git add *.txt` - добавить файлы с конкретным расширением в индекс
